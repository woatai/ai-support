## 当前阶段

当前项目处于第一阶段：先完成最小可运行闭环。

目标流程：

```text
用户在 Streamlit 页面输入问题
↓
Streamlit 调用 FastAPI /chat 接口
↓
FastAPI 调用大模型 API
↓
大模型根据客服 Prompt 生成回复
↓
FastAPI 返回 JSON
↓
Streamlit 展示 AI 回复
```

第一阶段暂不实现 RAG、订单查询和转人工规则，先把普通 AI 客服回复跑通。
